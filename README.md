# Cove Haven Quiz 🧠

An interactive quiz application built as the final project for the JavaScript course at Coderhouse.

## 🌐 Live Demo
[https://covehavenquiz.netlify.app/](https://covehavenquiz.netlify.app/)

## 📖 About
Cove Haven Quiz is a browser-based quiz app that challenges users with a series of questions. It dynamically loads questions from a JSON file and provides instant feedback on answers, tracking the user's score throughout the game.

## 🛠️ Built With
- JavaScript
- HTML5
- CSS3

## 📁 Project Structure
- `index.html` — Main page
- `script.js` — Quiz logic and DOM manipulation
- `index.css` — Styles
- `data.json` — Quiz questions and answers
- `images/` — Image assets

## 🚀 Getting Started
1. Clone the repository
   git clone https://github.com/ChrisEve14/projectofinalquiz.git
2. Open `index.html` in your browser — no installation needed!
